Getting Started

Up and running in minutes

1

Install

uvx chipiq-mcp>=0.0.9

Requires Python 3.13+

2

Configure

Add ChipIQ-MCP to your AI assistant's MCP configuration file

{
  "servers": {
    "chipiq-mcp-server": {
      "type": "stdio",
      "command": "uvx",
      "args": ["chipiq-mcp>=0.0.9"]
    }
  }
}
3

Ask Away

Start asking questions about your VCD files through your AI assistant

"Show me the module hierarchy in test.vcd"
